# Public Outputs

This repository includes selected non-sensitive figures derived from the AlphaPath
research programme. All included outputs satisfy the following conditions:

- no raw institutional data;
- no student-level records;
- no confidential identifiers;
- no private manuscript drafts;
- no internal file paths;
- no sensitive metadata.
